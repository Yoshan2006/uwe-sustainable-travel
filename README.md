# UWE Sustainable Travel Choices

## 📌 Project Overview
The **UWE Sustainable Travel Choices** application is an interactive, front-end dashboard designed to promote eco-friendly transportation options. By integrating map APIs and route planning capabilities, this project aims to help users visualize and select sustainable travel methods, reducing their carbon footprint. 

## 🚀 Features
* **Interactive Mapping:** Utilizes Leaflet.js to render responsive and interactive maps for route visualization.
* **Eco-Friendly Routing:** Integrates the OSRM (Open Source Routing Machine) API to calculate efficient travel paths.
* **User Dashboard:** A clean, dynamic dashboard built with modern JavaScript to track and display travel data.
* **Responsive Design:** Styled with CSS3 to ensure a seamless experience across desktop and mobile devices.

## 💻 Tech Stack
* **Front-end:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Libraries/APIs:** Leaflet.js, OSRM API

## ⚙️ Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project folder.
3. Open `index.html` (or `dashboard.html`) directly in your preferred web browser to view the application. *(No local server setup is required for the static front-end components).*
